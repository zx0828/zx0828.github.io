2分钟规则

zx_0828 2020-11-09 更多实用技巧 实用技巧文摘

每当你发现很难开始执行某项任务时，可以试试将其缩减成2分钟的版本。

  • 看一本书 → 看一页书
  • 写一篇文章 → 写一句话
  • 跑10公里 → 穿上跑鞋
  • 做100次俯卧撑 → 做1次俯卧撑
  • 多吃蔬菜水果 → 吃一个水果
  • 编写一个程序 → 编写一个函数 → 编写一行代码
